# Transit Accessibility App

## Overview
An accessibility-first transit app helping people with disabilities navigate public transport safely and sustainably.

## Problem Statement
Many public transit systems are inaccessible for people with disabilities. This project aims to provide real-time assistance, route planning, and accessibility information.

## Key Features
- Route planning for wheelchair users
- Accessibility alerts for each station/stop
- AI-powered assistance and recommendations
- Eco-friendly transit tracking

## Accessibility Focus
We prioritize inclusive design and provide features for:
- Wheelchair users
- Visually impaired
- Hearing impaired

## Tech Stack
- Frontend: React.js  
- Backend: Node.js / Python  
- AI/ML: TensorFlow / OpenAI API  
- Database: PostgreSQL
